Users

Οι χρήστες χωρίζονται σε 2 κατηγορίες, τους κανονικούς πελάτες-χρήστες και τους χρήστες με διαχειριστικά δικαιώματα.
Για το λόγο αυτό προστέθηκε και το πεδίο isAdmin στον πίνακα users.

Επίσης, οι οθόνες χωρίζονται σε 3 είδη, σύμφωνα με το ποιοι μπορούν να τις επισκεφθούν.
Εχουμε τις public, τις protected και τις admin.
Τις public μπορούν να τις επισκεφθούν και προσπελάσουν όλοι.
Τις protected μόνο όσοι είναι εγγεγραμμένοι στο σύστημα, έχουν γίνει δηλαδή registered, και έχουν κάνει login.
Τις admin μόνο όσοι έχουν τον αντίστοιχο ρόλο.

Οι απλοί χρήστες έχουν τη δυνατότητα να προσπελάσουν το λογαριασμό τους από την επιλογή MyAccount.

Ο admin user πέρα από το λογαριασμό του, έχει επιπλέον τη δυνατότητα να προσπελάσει την Admin area, όπου μπορεί να πραγματοποίησει 
διαφορες διαχειριστικες ενέργειες καθώς και στατιστικά queries.
Ο Admin μεταφέρεται στην admin area από την επιλογή MyAdmin.

Τα βασικά και πιο αντιπροσωπευτικά queries που λαμβάνουν χώρα στη βάση είναι τα εξής:
Τα γενικά queries όπου χρησιμοποιούνται για την παρουσίαση του περιέχομενου του eshop και κατά κύριο λόγο φιλτράρουν την πληροφορία βάσει των αναγκών του πελάτη.
Να σημειωθεί ότι τα γενικά queries είναι διαθέσιμα σε public οθόνες, ενώ λαμβάνουν ως εισόδους αρκετές παραμέτρους.
Τα προσωπικά queries, τα οποία χρησιμοποιούνται στις protected σελίδες λογαριασμών, όπου χρησιμοποιούνται από τον εκάστοτε χρήστη για να ενημερωθεί για τις προσωπικές του κινήσεις.
Τα στατιστικά queries, τα οποία είναι διαθέσιμα μόνο στην admin area και παρέχουν πληροφορία στον admin για το σύνολο της εφαρμογής.
Σε αυτές τις οθόνες, ο χρήστης μπορεί να επιλέξει το πλήθος των αποτελεσμάτων καθώς και την περίοδο που θέλει να εξετάσει.


Στη βασική οθόνη αναζήτησης, υπάρχουν 3 κατηγορίες φιλτραρίσματος οι οποίες ενεργούν συζευκτικά.
Αναζήτηση με κείμενο, λέξεις κλειδιά και search τύπου LIKE, με επιλογή κατηγοριών, οι οποίες είναι ΙΝ(σετ) και Price Range.
Επίσης, ο χρήστης έχει τη δυνατότητα για διάταξη των αποτελεσμάτων βάσει του πεδίου της επιλογής του.
